# Неделя 9, День 5 — Page Object Model

## Что такое POM
Это паттерн, где действия и локаторы страницы выносятся в отдельный класс.

## Зачем нужен POM
- меньше дублирования
- тесты читаются как сценарии
- проще менять локаторы

## Практика дня
Набросай один простой класс страницы с 2-3 действиями.
