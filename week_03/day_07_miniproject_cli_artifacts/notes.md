# Неделя 3, День 7 — Мини-проект: CLI-анализатор тестовых артефактов

## Цель дня
Собрать небольшую CLI-утилиту, которая читает файл с тестовыми событиями и выводит полезную сводку.

Темы недели, которые здесь закрепляются:
- исключения
- декораторы (минимально, если захочешь расширить)
- type hints
- регулярные выражения
- logging
- pathlib

## Идея проекта
Есть текстовый файл `artifacts.log` с разными сообщениями:
- `INFO`
- `WARNING`
- `ERROR`
- `TEST PASSED`
- `TEST FAILED`

Программа должна:
- прочитать файл,
- посчитать общее число строк,
- посчитать количество `ERROR`,
- посчитать количество `PASSED` и `FAILED`,
- найти все числовые id через regex,
- вывести краткую сводку в консоль.

## Что важно
- работать через `Path`
- разделить логику на функции
- не смешивать чтение файла, анализ и вывод в одном месте
- сделать понятную CLI-точку входа
