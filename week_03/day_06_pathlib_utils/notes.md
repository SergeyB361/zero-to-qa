# Неделя 3, День 6 — pathlib и утилиты

## Зачем нужен pathlib
`pathlib` — стандартный модуль Python для удобной работы с путями и файлами.

Он помогает:
- собирать пути без ручных слэшей,
- проверять существование файлов и папок,
- получать имя файла, расширение и родительскую папку,
- искать файлы по шаблону.

## Базовый пример
```python
from pathlib import Path

base_dir = Path(__file__).parent
file_path = base_dir / "data.txt"
print(file_path)
```

## Полезные свойства и методы
- `Path("file.txt")` — создать путь
- `path.exists()` — существует ли путь
- `path.is_file()` — это файл?
- `path.is_dir()` — это папка?
- `path.name` — имя файла
- `path.stem` — имя без расширения
- `path.suffix` — расширение
- `path.parent` — родительская папка
- `path.glob("*.txt")` — найти файлы по шаблону

## Полезные утилиты Python
Кроме `pathlib`, часто полезны простые встроенные утилиты:
- `enumerate()` — получить индекс и значение
- `zip()` — пройтись по двум спискам парами
- `sorted()` — отсортировать данные
- `any()` — проверить, есть ли хотя бы одно подходящее значение
- `all()` — проверить, что все значения подходят
