# Неделя 3, День 4 — Регулярные выражения

## Что такое regex
Регулярные выражения (`regex`) — это шаблоны для поиска текста.
В Python для этого используют модуль `re`.

Regex полезны, когда нужно:
- найти число в строке,
- извлечь email,
- проверить формат значения,
- разобрать лог или сообщение об ошибке.

## Базовые функции
- `re.search(pattern, text)` — найти первое совпадение
- `re.findall(pattern, text)` — найти все совпадения
- `re.sub(pattern, repl, text)` — заменить совпадения

## Частые шаблоны
- `\d` — цифра
- `\w` — буква/цифра/подчёркивание
- `.` — любой символ
- `+` — один или больше
- `*` — ноль или больше
- `?` — необязательный элемент

## Пример
```python
import re

text = "Order id: 12345"
match = re.search(r"\d+", text)
print(match.group())
```

## findall
```python
import re

text = "Errors: 404, 500, 403"
codes = re.findall(r"\d+", text)
print(codes)
```

## Что важно
- для шаблонов удобно использовать raw string: `r"..."`
- если совпадения нет, `re.search()` вернёт `None`
- тогда нельзя сразу вызывать `.group()` без проверки
