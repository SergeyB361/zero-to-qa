# Неделя 4, День 1 — Классы и объекты

## Что такое класс
Класс — это шаблон, по которому создаются объекты. Он описывает данные и поведение.

## Что такое объект
Объект — это конкретный экземпляр класса со своими значениями атрибутов.

## Базовый синтаксис
```python
class User:
    def __init__(self, name, age):
        self.name = name
        self.age = age

user = User("Ann", 25)
print(user.name)
```

## Важные понятия
- `class` — объявление класса
- `__init__` — метод, который вызывается при создании объекта
- `self` — ссылка на текущий объект
- атрибуты — данные объекта, например `self.name`

## Практические правила
- Имена классов пиши в `CamelCase`: `User`, `TestCase`, `ApiClient`.
- Атрибуты экземпляра обычно создаются внутри `__init__`.
- Через `self` объект хранит собственное состояние.
