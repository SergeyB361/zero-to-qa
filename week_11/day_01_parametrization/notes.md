# Неделя 11, День 1 — Параметризация

## Что это такое
`@pytest.mark.parametrize` позволяет запускать один и тот же тест на разных наборах данных.

## Зачем это нужно
- убрать дублирование тестов
- быстро покрыть несколько сценариев
- сделать данные теста явными

## Базовый пример
```python
import pytest

@pytest.mark.parametrize("value, expected", [
    (2, True),
    (3, False),
])
def test_is_even(value, expected):
    assert (value % 2 == 0) is expected
```
