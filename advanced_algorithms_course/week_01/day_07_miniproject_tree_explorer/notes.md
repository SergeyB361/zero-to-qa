# Неделя 1, День 7 - Мини-проект: Tree Explorer

## Цель дня
Собрать маленький инструмент для обхода дерева и анализа его свойств.

## Идея проекта
У тебя есть бинарное дерево.
Нужно уметь:
- считать сумму значений;
- считать количество узлов;
- считать высоту дерева;
- делать хотя бы один DFS-обход.

## Что важно
- использовать рекурсию по смыслу;
- не дублировать логику;
- отдельно держать функции анализа дерева.

Открой `spec.md` и реализуй проект в `tree_explorer.py`.
