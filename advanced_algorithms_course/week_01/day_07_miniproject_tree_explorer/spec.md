# ТЗ: Tree Explorer

Реализуй проект в `tree_explorer.py`.

## Что должно быть в MVP
1. Класс `TreeNode`
2. `tree_sum(root)`
3. `tree_size(root)`
4. `tree_height(root)`
5. `preorder(root)`
6. `main()` с демонстрацией на одном дереве

## Пример дерева
Можно использовать такое:

```text
        10
       /        5    20
          /           15   30
```

## Пример допустимого вывода
```text
Sum: 80
Size: 5
Height: 3
Preorder: [10, 5, 20, 15, 30]
```

## Критерии готовности
- файл запускается без ошибок;
- функции работают от `TreeNode`, а не от списка;
- рекурсия используется осмысленно;
- вывод читается без открытия кода.
