# Курс: Алгоритмы и структуры данных — продвинутый уровень

Продвинутый курс по рекурсии, связным структурам, деревьям, heap, графам и смешанным алгоритмическим задачам.

## Зачем нужен продвинутый курс
- перейти от базовых паттернов к более абстрактным структурам данных;
- научиться решать задачи на деревья, графы и поиск по состояниям;
- подготовиться к более сложным interview-style задачам;
- собрать продвинутый algorithmic toolkit.

## Формат
- `3` недели
- `7` дней в неделе
- обычный день: `notes.md`, `examples.py`, `practice.py`
- день мини-проекта: `notes.md`, `spec.md`, основной `.py` файл

## Структура курса
- `week_01/` — Рекурсия, связные структуры и деревья
- `week_02/` — Heap и графы
- `week_03/` — Продвинутые паттерны и итоговый toolkit

## Связанные файлы
- `ROADMAP.md` — полный план по неделям и дням

## Как проходить курс
1. сначала читать `notes.md`;
2. затем запускать `examples.py`;
3. потом решать `practice.py`;
4. в конце недели делать мини-проект.
