# ТЗ: Финальный мини-проект: Algorithmic Toolkit

Реализуй проект в `algorithmic_toolkit.py`.

## Что должно быть в MVP
Минимум 5 функций из разных блоков курса. Например:
1. `has_duplicate(items)`
2. `binary_search(items, target)`
3. `merge_intervals(intervals)`
4. `top_k_largest(items, k)`
5. `bfs_levels(graph, start)`

## Обязательные требования
- функции должны быть из разных паттернов, а не из одной темы;
- проект должен запускаться через `main()`;
- в `main()` нужно показать минимум по одному примеру на каждую функцию;
- вывод должен быть понятным без чтения кода.

## Пример допустимого вывода
```text
Has duplicate: True
Binary search index: 3
Merged intervals: [(1, 6), (8, 10)]
Top 3 largest: [9, 8, 7]
BFS levels: {'A': 0, 'B': 1, 'C': 1, 'D': 2}
```

## Критерии готовности
- файл запускается без ошибок;
- код разбит на функции;
- проект действительно демонстрирует разные классы алгоритмов;
- `main()` работает как demo, а не как хаотичный набор print-ов.
