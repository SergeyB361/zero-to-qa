# Неделя 3, День 6 - Подготовка финального проекта

## Цель дня
Подготовить финальный мини-проект недели и собрать уже пройденные паттерны в один связный toolkit.

## Что должно войти в проект
Минимально стоит включить:
- один hash-based алгоритм;
- один search/sort алгоритм;
- один graph-алгоритм;
- один utility для ranges/windows или числовых задач.

## Принцип сборки
Не надо пытаться впихнуть всё.
Лучше сделать небольшой, но связный набор функций.

## Что важно
- каждая функция решает одну задачу;
- есть понятный `main()`;
- demo flow показывает, зачем toolkit вообще нужен;
- код читается без открытия каждой темы отдельно.

## Что дальше
Открой `practice.py` и подготовь shortlist функций для финального проекта.
Потом переходи к `spec.md` мини-проекта следующего дня.
