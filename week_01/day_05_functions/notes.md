# Неделя 1, День 5 — Функции: основы

## Профессиональное определение
Функция — это именованный, переиспользуемый программный объект (подпрограмма), который инкапсулирует набор инструкций, принимает входные аргументы, выполняет вычисление в собственной локальной области видимости и возвращает результат (или побочный эффект).

В Python функция является объектом первого класса: ее можно передавать как значение, хранить в структурах данных и возвращать из других функций.

## Что такое функция
Функция — это именованный блок кода, который выполняет задачу.

## Базовый синтаксис
```python
def function_name(param1, param2):
    # тело функции
    return result
```

## Ключевые идеи
- `def` объявляет функцию.
- Параметры — входные данные.
- `return` возвращает результат.
- Без `return` функция возвращает `None`.

## Мини-примеры
```python
def add(a, b):
    return a + b

print(add(2, 3))  # 5
```

```python
def greet(name):
    print(f"Привет, {name}!")

greet("Sergey")
```

## Scope (область видимости)
Переменные внутри функции локальные и не видны снаружи.
