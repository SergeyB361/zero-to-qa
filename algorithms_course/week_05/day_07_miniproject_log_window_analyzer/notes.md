# Неделя 5, День 7 - Мини-проект: Log Window Analyzer

## Цель дня
Собрать мини-инструмент, который анализирует последние события лога через `deque` и sliding window.

## Идея проекта
Есть поток лог-уровней, например:
- `INFO`
- `WARNING`
- `ERROR`

Нужно поддерживать окно последних `k` записей и уметь быстро:
- показывать текущее окно;
- считать количество `ERROR` в окне;
- считать количество `WARNING` в окне.

## Что важно
- использовать `deque(maxlen=k)`;
- не пересчитывать всё через громоздкую логику;
- разделить код на функции;
- сделать понятный demo flow.

Открой `spec.md` и реализуй проект в `log_window_analyzer.py`.
