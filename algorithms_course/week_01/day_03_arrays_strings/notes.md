# Неделя 1, День 3 - Массивы, списки и строки

## Что важно в этой теме
В Python вместо классического массива чаще всего используют `list`.
Для алгоритмов это одна из базовых структур.

Строка тоже ведёт себя как последовательность, но она неизменяема.

## Список
Список хранит упорядоченные элементы.

Полезно помнить:
- доступ по индексу обычно `O(1)`;
- проход по списку `O(n)`;
- вставка в середину и удаление из середины обычно `O(n)`.

## Индексы
- `items[0]` - первый элемент;
- `items[-1]` - последний элемент.

## Строка
Строка - это последовательность символов.

Но строку нельзя менять по месту.
Обычно создаётся новая строка.

## Что важно для алгоритмов
- уметь проходить по элементам;
- уметь использовать индексы;
- понимать стоимость вставки и удаления;
- помнить, что строки immutable.

## Где это встречается
- задачи на палиндром;
- разворот строки;
- подсчёт символов;
- поиск элемента по индексу;
- обработка массивов данных.

## Типичные ошибки
1. Путать список и строку.
2. Забывать про выход за границы индекса.
3. Пытаться менять символ строки напрямую.
4. Игнорировать стоимость вставки в начало списка.
