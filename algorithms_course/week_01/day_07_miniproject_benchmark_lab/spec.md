# ТЗ: Benchmark Lab

Реализуй проект в `benchmark_lab.py`.

## Цель
Сравнить скорость нескольких подходов к поиску элемента.

## Минимальные требования
1. Реализуй функцию `linear_contains(items, target)`.
2. Реализуй функцию `set_contains(items_set, target)`.
3. Реализуй функцию `measure_average(func, repeats)` на базе `time.perf_counter()`.
4. Подготовь данные хотя бы трёх размеров:
   - `1_000`
   - `10_000`
   - `100_000`
5. Для каждого размера выведи понятный отчёт.

## Формат результата
Пример допустимого вывода:

```text
=== n = 1000 ===
linear_contains: 0.012345
set_contains:    0.000031
```

## Опционально
- добавить бинарный поиск по отсортированному списку;
- сделать несколько target-значений;
- посчитать среднее по нескольким прогонам.

## Критерии готовности
- проект запускается без ошибок;
- сравнение выполняется на одинаковых данных;
- код разбит на функции;
- вывод читается без открытия кода.
