# Курс: Алгоритмы и структуры данных

Отдельный курс по алгоритмическому мышлению, структурам данных и базовым паттернам решения задач.

## Зачем этот курс нужен
- научиться оценивать сложность решений;
- выбирать подходящую структуру данных под задачу;
- уверенно разбирать типовые алгоритмические паттерны;
- получить базу для backend, QA automation, интервью и pet project-задач.

## Формат
- `8` недель
- `7` дней в неделе
- обычный день: `notes.md`, `examples.py`, `practice.py`
- день мини-проекта: `notes.md`, `spec.md`, основной `.py` файл

## Структура курса
- `week_01/` — Введение в алгоритмы
- `week_02/` — Числовые алгоритмы
- `week_03/` — Поиск и сортировки
- `week_04/` — Hash-based подходы
- `week_05/` — Stack, Queue, Deque
- `week_06/` — Рекурсия, связные структуры и деревья
- `week_07/` — Heap и графы
- `week_08/` — Итоговая алгоритмическая неделя

## Связанные файлы
- `ROADMAP.md` — полный план по неделям и дням

## Как проходить курс
1. сначала читать `notes.md`;
2. затем запускать `examples.py`;
3. потом решать `practice.py`;
4. в конце недели делать мини-проект.
