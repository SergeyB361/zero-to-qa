# ТЗ: Мини-проект: Tree Explorer

Реализуй проект в `tree_explorer.py`.

## Минимальные требования
- использовать темы текущей недели;
- разбить решение на функции;
- добавить демонстрационный запуск через `main()`;
- показать хотя бы 2-3 сценария работы.

## Что проверить
- код запускается без ошибок;
- результат можно проверить глазами по выводу;
- логика не зашита в один длинный блок.
