# Неделя 5, День 4 — AAA-паттерн

## Что такое AAA
AAA означает:
- `Arrange` — подготовка данных
- `Act` — выполнение действия
- `Assert` — проверка результата

## Зачем это нужно
- тесты легче читать
- проще понимать причину падения
- меньше хаоса в структуре теста

## Пример
```python
def test_discount():
    # Arrange
    price = 100

    # Act
    result = price * 0.9

    # Assert
    assert result == 90
```
