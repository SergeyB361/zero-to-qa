# ТЗ: Набор тестов для функций из недель 1–2

Реализуй мини-проект в этой папке.

## Что сделать
1. Создай файл `app.py` с 3-5 простыми функциями.
2. Возьми идеи из тем недель 1–2:
- числа
- условия
- строки
- списки
- словари

## Примеры функций
- `is_even(number)`
- `get_initials(first_name, last_name)`
- `get_max(numbers)`
- `is_adult(age)`

## Что проверить тестами
- корректный результат на обычных данных
- граничные значения
- минимум 1 негативный сценарий, если он уместен

## Минимальный состав файлов
- `app.py`
- `test_app.py`

## Цель
Получить первый законченный набор автотестов на `pytest`.
