# Неделя 5, День 6 — Запуск и отчёты

## Полезные команды PyTest
- `pytest` — запуск всех тестов
- `pytest -v` — подробный вывод
- `pytest -k name` — запуск тестов по части имени
- `pytest --tb=short` — короткий traceback

## Зачем это нужно
- быстрее запускать нужные тесты
- проще читать результат падения
- удобнее локализовать проблему

## Allure
Allure используют для более наглядных отчётов по тестам.
На этом этапе важно понимать назначение инструмента и базовый сценарий запуска.

## Практика дня
Открой `practice.py` и запиши команды, которые ты бы использовал для запуска нужных тестов.
