# Неделя 5, День 5 — Фикстуры: основы

## Что такое fixture
Фикстура в `pytest` помогает подготовить данные или объекты для тестов.

## Зачем нужны фикстуры
- убрать дублирование
- централизовать подготовку данных
- сделать тесты чище

## Пример
```python
import pytest

@pytest.fixture
def user_data():
    return {"username": "anna", "role": "admin"}
```

## Где это полезно
- тестовые пользователи
- временные файлы
- объекты API-клиента
- конфигурация окружения

## Практика дня
Открой `practice.py` и подготовь функции, для которых удобно переиспользовать входные данные.
