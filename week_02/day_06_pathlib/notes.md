# Неделя 2, День 6 — pathlib

## Что такое pathlib
`pathlib` — это удобный модуль стандартной библиотеки Python для работы с путями и файлами.

Он помогает:
- собирать пути без ручных слэшей,
- проверять существование файлов,
- читать имена и расширения,
- работать с директориями понятнее, чем через строки.

## Базовый пример
```python
from pathlib import Path

base_dir = Path(__file__).parent
file_path = base_dir / "data.txt"
print(file_path)
```

## Полезные методы и свойства
- `Path("file.txt")` — создать путь
- `path.exists()` — существует ли путь
- `path.is_file()` — это файл?
- `path.is_dir()` — это папка?
- `path.name` — имя файла
- `path.stem` — имя без расширения
- `path.suffix` — расширение
- `path.parent` — родительская папка

## Почему это важно
После тем про файлы и JSON `pathlib` делает работу с путями надёжнее и чище.
Это особенно полезно в тестах, CLI-утилитах и мини-проектах.
