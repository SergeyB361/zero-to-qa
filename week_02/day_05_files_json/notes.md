## Файлы в Python
Работа с файлами делается через `open()`.

```python
with open("data.txt", "r", encoding="utf-8") as f:
    text = f.read()
```

Почему `with`:
- файл автоматически закрывается,
- меньше шансов утечки ресурсов.

## Режимы открытия
- `r` — чтение (файл должен существовать)
- `w` — запись (перезаписывает файл)
- `a` — добавление в конец
- `x` — создать новый файл (ошибка, если уже есть)

## Полезные методы
- `read()` — прочитать всё
- `readline()` — прочитать одну строку
- `readlines()` — список строк
- `write(text)` — записать строку

## JSON
JSON (JavaScript Object Notation) — это простой текстовый формат обмена данными.
Его удобно читать человеку и программе, поэтому JSON часто используют в API, конфиг-файлах и автотестах.

Что важно понимать:
- JSON хранит структуру данных: объекты (как словари), массивы (как списки), строки, числа, `true/false`, `null`.
- В Python после чтения JSON превращается в обычные `dict` и `list`.

В Python:
- `json.load(file)` — читать JSON из файла
- `json.dump(data, file, ensure_ascii=False, indent=2)` — писать JSON в файл

```python
import json

with open("user.json", "r", encoding="utf-8") as f:
    user = json.load(f)

print(user["name"])
```

## Частые ошибки
- забыли `encoding="utf-8"`
- перепутали `load` и `loads` (`load` — из файла, `loads` — из строки)
- забыли обработать отсутствие файла (`FileNotFoundError`)
