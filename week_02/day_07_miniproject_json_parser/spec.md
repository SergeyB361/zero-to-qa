# ТЗ: Консольный JSON-парсер

Реализуй в `json_parser.py`:

1. Данные
- Создай файл `users.json` в папке проекта.
- В файле должен быть список словарей:

```json
[
  {"name": "Alice", "age": 25, "active": true},
  {"name": "Bob", "age": 31, "active": false},
  {"name": "Charlie", "age": 29, "active": true}
]
```

2. Функции
- `load_users(path)`
  - читает JSON из файла
  - возвращает список пользователей

- `get_names(users)`
  - возвращает список имен всех пользователей

- `get_active_users(users)`
  - возвращает список словарей только для активных пользователей

- `get_average_age(users)`
  - возвращает средний возраст пользователей

3. Основной сценарий
- загрузить пользователей из `users.json`
- вывести:
  - `Total users: <число>`
  - `Names: <список имен>`
  - `Active users: <число>`
  - `Average age: <число>`

4. Точка входа
- `if __name__ == "__main__":`
  - вызов `main()`

Опционально:
- вывести имена только активных пользователей отдельной строкой.
