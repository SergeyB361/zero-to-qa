# Неделя 2, День 6 — Множества

## Что такое set
`set` — это множество уникальных элементов.

Главные свойства:
- элементы не повторяются,
- порядок элементов не гарантируется,
- удобно проверять наличие значения,
- удобно убирать дубликаты.

## Создание множества
```python
numbers = {1, 2, 3}
letters = set(["a", "b", "a"])
print(letters)
```

## Полезные методы
- `add(x)` — добавить элемент
- `remove(x)` — удалить элемент, если его нет, будет ошибка
- `discard(x)` — удалить элемент без ошибки
- `pop()` — удалить случайный элемент
- `clear()` — очистить множество

## Полезные операции
- `a | b` — объединение
- `a & b` — пересечение
- `a - b` — разность
- `x in s` — проверка наличия

## Где это полезно
После списков и словарей множества помогают:
- быстро убирать дубликаты,
- искать общие элементы,
- проверять принадлежность значений.
