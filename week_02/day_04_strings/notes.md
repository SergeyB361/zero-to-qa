# Неделя 2, День 4 — Строки

## Что такое строка
`str` — неизменяемая последовательность символов.

## Базовые операции
- длина: `len(s)`
- индексы/срезы: `s[0]`, `s[1:4]`
- конкатенация: `a + b`
- повторение: `"-" * 10`

## Полезные методы
- `lower()` — переводит строку в нижний регистр.
- `upper()` — переводит строку в верхний регистр.
- `title()` — делает первую букву каждого слова заглавной.
- `strip()` — удаляет пробелы (и другие символы) по краям строки.
- `replace(old, new)` — заменяет все вхождения `old` на `new`.
- `split(sep)` — разбивает строку на список по разделителю `sep`.
- `join(iterable)` — склеивает список строк в одну строку через разделитель.
- `startswith(prefix)` — проверяет, начинается ли строка с `prefix`.
- `endswith(suffix)` — проверяет, заканчивается ли строка на `suffix`.
- `find(sub)` — возвращает индекс первого вхождения `sub` (или `-1`, если не найдено).

## Частые сценарии QA
- нормализация текста (`lower().strip()`)
- проверка подстрок в логах
- разбор CSV/простых строк через `split`

## Пример
```python
line = "  ERROR 404: Not Found  "
clean = line.strip().lower()
print(clean)  # "error 404: not found"
```
