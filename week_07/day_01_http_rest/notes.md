# Неделя 7, День 1 — HTTP и REST

## Базовые HTTP-методы
- `GET` — получить данные
- `POST` — создать
- `PUT` — обновить полностью
- `PATCH` — обновить частично
- `DELETE` — удалить

## Что важно в API-тестах
- статус-код
- заголовки
- тело ответа
- время ответа

## REST
REST обычно предполагает понятные ресурсы и предсказуемые URL.
