# Неделя 7, День 4 — Валидация данных

## Зачем валидировать JSON
Даже если статус-код `200`, ответ может быть неполным или неправильным.

## Что проверяют
- обязательные поля
- типы значений
- вложенные структуры
- JSON Schema

## Практика дня
Подготовь проверки ответа API не только по одному значению, но и по структуре.
