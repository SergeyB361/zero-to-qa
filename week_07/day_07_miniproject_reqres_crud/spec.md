# ТЗ: CRUD API-тесты для reqres.in

## Что нужно покрыть
- `GET` список пользователей
- `GET` одного пользователя
- `POST` создание пользователя
- `PUT` или `PATCH` обновление
- `DELETE` удаление

## Что проверять
- статус-коды
- ключевые поля ответа
- базовую структуру JSON

## Минимальный состав файлов
- `api_client.py`
- `test_users_api.py`
