# ТЗ: Мини-проект — генерация и проверка тестовых данных

## Исходный файл
Используй:
- data_targets.py

## Что нужно сделать
Построй мини-набор инструментов для тестовых данных:
1. factory для пользователя
2. builder для заказа
3. reusable datasets для ролей и статусов
4. edge наборы для суммы и длины имени
5. простые проверки целостности данных

## Обязательные требования
- минимум две factory
- минимум один builder
- минимум два reusable dataset
- минимум два edge-набора
- минимум три явные проверки-инварианта

## Что должно быть в решении
- factories.py
- builders.py
- datasets.py
- rules.py
- при желании demo.py

## Критерии качества
- понятные names
- предсказуемые default values
- отсутствие случайности без контроля
- повторное использование данных без giant JSON
