# Bonus Block — Advanced Python OOP

Это дополнительный блок для углубления Python OOP.

Он не входит в основной прогресс `advanced_course/week_01`-`week_10`.

Когда проходить:
- после завершения базовой `Week 4 — ООП`
- лучше после основных QA-недель, когда уже есть практический контекст

Что здесь будет:
- `__str__` и `__repr__`
- `__eq__` и `__hash__`
- `__call__`
- дескрипторы
- `__getattr__`, `__getattribute__`, `__setattr__`
- `dataclasses`, `multiple inheritance`, `__slots__`
- бонусный мини-проект
