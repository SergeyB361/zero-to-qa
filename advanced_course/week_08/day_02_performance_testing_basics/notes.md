# Senior QA Engineering, День 2 — Performance testing basics

## Зачем это нужно
- Даже лёгкий performance smoke помогает поймать грубые деградации до релиза.
- QA должен уметь отличать latency, throughput и stability.
- Performance mindset полезен даже без тяжёлого load testing.

## Что важно понять
- Latency отвечает за время ответа.
- Throughput показывает объём обработки за единицу времени.
- Performance smoke не заменяет полноценный нагрузочный тест, но даёт быстрый сигнал.

## Частые ошибки
- Делать вывод о производительности по одному случайному прогону.
- Смешивать функциональный сбой и нагрузочную деградацию.
- Игнорировать warm-up и шум окружения.
