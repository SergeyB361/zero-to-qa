# Неделя 5, День 4 — AAA-паттерн

## Что такое AAA
AAA означает:
- `Arrange` — подготовка данных
- `Act` — выполнение действия
- `Assert` — проверка результата

## Зачем это нужно
- тесты легче читать
- проще понимать причину падения
- меньше хаоса в структуре теста

## Пример
```python
def test_discount():
    # Arrange
    price = 100

    # Act
    result = price * 0.9

    # Assert
    assert result == 90
```

## Как думать через AAA
- сначала подготовь входные данные
- потом выполни одно действие
- потом проверь результат

Если в тесте перемешано всё сразу, его тяжелее читать и поддерживать.

## Практические правила
- в тесте должен быть один явный `Act`
- подготовку данных лучше не смешивать с проверками
- AAA особенно полезен даже в коротких тестах: он дисциплинирует структуру
