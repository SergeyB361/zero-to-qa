# Неделя 6, День 7 — Мини-проект: CRUD API tests

## Цель дня
Собрать небольшой API-набор тестов на CRUD-сценарии и закрепить всю неделю.

## Что здесь закрепляется
- HTTP-методы
- requests
- проверки статус-кодов
- валидация JSON
- организация тестов

## Как подойти к проекту
1. выбери понятные CRUD-сценарии
2. сначала добейся стабильных запросов
3. потом добавь ясные проверки
4. в конце приведи код к понятной структуре

## Что важно
- названия тестов должны объяснять сценарий
- проверки должны быть конкретными
- лучше меньше тестов, но они понятные и рабочие
