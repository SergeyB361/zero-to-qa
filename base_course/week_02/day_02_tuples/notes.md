# Неделя 2, День 2 — Кортежи

## Что такое кортеж
Кортеж (`tuple`) — упорядоченная, но неизменяемая коллекция элементов.

```python
point = (10, 20)
```

## Когда использовать tuple
- когда данные не должны меняться
- для фиксированных наборов значений (координаты, настройки, пары ключ-значение)
- как ключи словаря (если элементы тоже неизменяемые)

## Базовые операции
- индекс: `t[0]`
- срез: `t[1:3]`
- длина: `len(t)`
- проверка: `x in t`
- подсчет: `t.count(x)`
- позиция: `t.index(x)`

## Важно про неизменяемость
```python
t = (1, 2, 3)
# t[0] = 100  # TypeError
```

## Tuple packing / unpacking
```python
person = ("Sergey", 28, "QA")
name, age, role = person
print(name, age, role)
```

## tuple с одним элементом
```python
a = (5,)   # правильно
b = (5)    # это int, не tuple
```
