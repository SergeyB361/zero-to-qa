# Неделя 2, День 7 — Мини-проект: Парсер JSON

## Цель дня
Собрать небольшой консольный проект, который читает JSON-файл и выводит по нему полезную сводку.

Темы, которые здесь закрепляются:
- словари и списки
- строки
- файлы
- JSON

## Идея проекта
Есть файл `users.json` со списком пользователей.
Программа должна:
- прочитать JSON,
- посчитать количество пользователей,
- вывести имена,
- отдельно показать только активных пользователей,
- посчитать средний возраст.

## Что важно
- работать именно с данными из файла,
- писать код небольшими понятными шагами,
- разбить логику на функции.

## Следующий шаг
Открой `spec.md` и реализуй требования в `json_parser.py`.
