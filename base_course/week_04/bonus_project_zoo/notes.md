# Неделя 4 — Дополнительный сквозной проект: Зоопарк

## Что это
`Зоопарк` — необязательный проект на всю `Week 4`.

Это не задание только для `Day 1`. Проект специально лучше делать по мере прохождения недели, потому что в нём используются темы сразу нескольких дней:
- классы и объекты
- методы
- наследование
- инкапсуляция
- полиморфизм

## Как проходить проект по неделе
- после `Day 1`: создай базовый класс `Animal` и его атрибуты
- после `Day 2`: добавь методы `eat()`, `play()`, `status()`
- после `Day 3`: сделай `Lion`, `Parrot`, `Turtle`
- после `Day 4`: аккуратно оформи работу с энергией и доступ к данным
- после `Day 5`: доведи полиморфизм и класс `Zoo`
- в конце `Week 4`: собери итоговый рабочий сценарий

## Что открыть
- условия проекта: `spec.md`
- код: `zoo_project.py`

## Как проверять себя
- каждый шаг должен запускаться без ошибок
- не делай всё сразу одним большим коммитом в голове
- лучше двигаться по шагам и проверять проект после каждого блока
