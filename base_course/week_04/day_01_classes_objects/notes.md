# Неделя 4, День 1 — Классы и объекты

## Зачем это нужно
ООП помогает объединять данные и поведение в одной сущности. Для QA это полезно, когда нужно описывать пользователей, тест-кейсы, API-клиенты, страницы UI и другие объекты предметной области.

## Что такое класс
Класс — это шаблон. Он описывает, какие данные будут храниться в объекте и какие действия объект умеет выполнять.

## Что такое объект
Объект — это конкретный экземпляр класса. У разных объектов одного и того же класса могут быть разные значения атрибутов.

Пример:
- класс `User` — шаблон пользователя
- объект `user_1` — пользователь `Ann`
- объект `user_2` — пользователь `Sergey`

## Атрибуты объекта
Атрибуты — это данные, которые принадлежат конкретному объекту.

Пример:
- `user.name`
- `user.age`

Если создать два объекта, их атрибуты будут храниться отдельно.

## Базовый синтаксис
```python
class User:
    def __init__(self, name, age):
        self.name = name
        self.age = age

user = User("Ann", 25)
print(user.name)
```

## Как создаётся объект пошагово
Когда ты пишешь:

```python
user = User("Ann", 25)
```

происходит следующее:
1. Python создаёт новый объект класса `User`
2. затем автоматически вызывает `__init__`
3. в `self` попадает созданный объект
4. `name` получает значение `"Ann"`
5. `age` получает значение `25`
6. `self.name` и `self.age` сохраняются внутри объекта

## Что такое `self`
`self` — это ссылка на текущий объект.

Важно:
- `self` не передают вручную при обычном вызове
- Python делает это автоматически

То есть:
```python
user = User("Ann", 25)
```
выглядит как обычный вызов, но внутри Python связывает созданный объект с `self`.

## Что такое `__init__`
`__init__` — это специальный метод, который вызывается при создании объекта. Обычно в нём задают начальное состояние объекта.

## Важные понятия
- `class` — объявление класса
- объект — экземпляр класса
- `__init__` — инициализация объекта
- `self` — текущий объект
- атрибуты — данные объекта, например `self.name`

## Практические правила
- Имена классов пиши в `CamelCase`: `User`, `TestCase`, `ApiClient`.
- Атрибуты экземпляра обычно создаются внутри `__init__`.
- Через `self` объект хранит собственное состояние.
- Сегодня фокус только на создании объектов и хранении данных. Методы объектов будут подробно на следующем уроке.
