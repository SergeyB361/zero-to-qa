# Неделя 7, День 1 — Основы Playwright

## Что такое Playwright
Playwright — это инструмент для автоматизации браузера. Он позволяет открывать страницу, искать элементы, выполнять действия и проверять результат.

## Почему это важно для QA
UI-тесты позволяют проверять пользовательские сценарии на уровне интерфейса:
- открытие страниц
- вход в систему
- заполнение форм
- навигацию по сайту

## Что обычно есть в первом тесте
- запуск браузера
- создание страницы
- переход по URL
- простая проверка результата

## Ключевые понятия
- `browser` — браузер
- `page` — вкладка/страница
- `locator` — способ найти элемент
- `expect` — проверка ожидаемого результата

## Практические правила
- сначала научись писать простой и стабильный тест
- не усложняй первый сценарий множеством действий
- UI-тесты медленнее API и unit, поэтому особенно важна понятность
