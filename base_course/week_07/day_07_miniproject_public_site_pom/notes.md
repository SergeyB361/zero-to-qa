# Неделя 7, День 7 — Мини-проект: UI-тесты публичного сайта с POM

## Цель дня
Собрать небольшой UI-проект и закрепить всю неделю Playwright.

## Что здесь закрепляется
- локаторы
- действия
- assertions
- Page Object Model
- структура UI-теста

## Как подойти к проекту
1. сначала выбери простой публичный сценарий
2. добейся стабильного прохождения вручную через Playwright
3. затем выдели page object
4. только потом добавляй финальные проверки и чистку структуры

## Что важно
- сценарий должен быть реалистичным, но не слишком большим
- page object должен упрощать тест, а не усложнять его
- тест должен быть читаемым и воспроизводимым

## Следующий шаг
Открой `spec.md` и реализуй проект в коде.
