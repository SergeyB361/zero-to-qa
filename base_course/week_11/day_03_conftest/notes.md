# Неделя 11, День 3 — conftest.py

## Что такое conftest.py
`conftest.py` — специальный файл `pytest`, где удобно хранить общие фикстуры и настройки для группы тестов.

## Зачем это нужно
- не дублировать фикстуры по многим файлам
- держать общую тестовую инфраструктуру в одном месте
- упрощать структуру проекта

## Что туда часто кладут
- общие фикстуры
- hooks и базовые настройки
- вспомогательную тестовую конфигурацию

## Практические правила
- в `conftest.py` лучше хранить общие вещи, а не случайную бизнес-логику
- чем ближе файл к тестам, тем понятнее его зона ответственности
- не нужно превращать один `conftest.py` в огромный свалочный файл
