# ТЗ: Консольный калькулятор

Реализуй в `calculator.py`:

1) Функции операций
- `add(a, b)` -> сумма
- `sub(a, b)` -> разность
- `mul(a, b)` -> произведение
- `div(a, b)` ->
  - если `b == 0`, вернуть строку `"Division by zero"`
  - иначе вернуть результат деления

2) Ввод числа
- `read_number(prompt)`
- внутри `while True` + `try/except ValueError`
- возвращает `float`

3) Основной цикл
- показать меню:
  - `+` сложение
  - `-` вычитание
  - `*` умножение
  - `/` деление
  - `q` выход
- если операция не из списка — вывести `"Unknown operation"` и продолжить
- для операций запрашивать 2 числа через `read_number`
- выводить результат как `Result: <value>`

4) Точка входа
- `if __name__ == "__main__":`
  - вызов `main()`

Опционально:
- округление float до 2 знаков при выводе.
