# Неделя 8, День 1 — SQL: основы

## Что такое SQL
SQL — это язык для работы с данными в реляционных базах.

## Основные операции
- `SELECT` — получить данные
- `INSERT` — добавить данные
- `UPDATE` — изменить данные
- `DELETE` — удалить данные

## Частые части запроса
- `FROM` — из какой таблицы читать
- `WHERE` — фильтрация
- `ORDER BY` — сортировка

## Пример
```sql
SELECT name, age
FROM users
WHERE age >= 18;
```

## Почему это важно для QA
- можно проверять данные после действий приложения
- можно готовить данные для тестов
- можно верифицировать интеграции

## Практические правила
- всегда понимай, какую таблицу и какие строки ты читаешь
- `WHERE` особенно важен: без него можно изменить или удалить слишком много данных
- сначала научись простым понятным запросам, потом усложняй
