# Неделя 8, День 2 — SQLite в Python

## Что такое sqlite3
`sqlite3` — встроенный модуль Python для работы с SQLite-базой.

## Базовые шаги
1. подключиться к базе
2. создать cursor
3. выполнить SQL-запрос
4. получить результат
5. закрыть соединение

## Пример
```python
import sqlite3

conn = sqlite3.connect("test.db")
cur = conn.cursor()
cur.execute("SELECT name FROM users")
rows = cur.fetchall()
conn.close()
```

## Где это полезно
- проверка данных после API- или UI-действий
- подготовка тестовых данных
- очистка данных после теста

## Практические правила
- после изменений не забывай про `commit()`, если это нужно
- соединение лучше закрывать аккуратно
- в тестах полезно понимать, какие данные ты читаешь и зачем
