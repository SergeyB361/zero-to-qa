# Неделя 8, День 3 — Allure: углубленно

## Зачем нужен Allure
Делает результаты тестов нагляднее: шаги, вложения, история, severity.

## Что часто используют
- `@allure.step`
- `@allure.title`
- `allure.attach`
- `@allure.severity`

## Практика дня
Продумай, что ты бы прикладывал к отчету для API и UI тестов.
