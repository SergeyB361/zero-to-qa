# Неделя 8, День 7 — Мини-проект: API + Allure + SQLite

## Цель дня
Собрать маленький интегрированный проект, где соединяются API-проверки, отчётность и проверка данных.

## Что здесь закрепляется
- API-тесты
- проверки данных
- SQLite
- Allure
- структура тестового мини-проекта

## Как подойти к проекту
1. сначала собери рабочий API-сценарий
2. затем добавь понятные проверки
3. потом подключи базу или симуляцию проверки данных
4. в конце оформи шаги и отчётность

## Что важно
- не пытайся сделать слишком большой проект
- важнее связность и понятность, чем количество файлов
- каждый слой должен иметь свою роль
