# Неделя 8, День 4 — Git для QA

## Базовые команды
- `git status`
- `git add`
- `git commit`
- `git push`
- `git pull`
- `git checkout -b`

## Что важно QA-инженеру
- понимать ветки
- делать понятные коммиты
- не терять контекст изменений

## Практика дня
Подготовь свой маленький рабочий сценарий от изменения файла до push.
