# Неделя 6, День 4 — Моки: основы

## Что такое mock
Mock заменяет реальную зависимость управляемым объектом.

## Когда используют mock
- внешний API недоступен
- не хочется делать реальный запрос
- нужно проверить вызов зависимости

## Инструменты
- `unittest.mock.Mock`
- `MagicMock`
- `patch`

## Практика дня
Подготовь маленькую функцию, которая вызывает внешнюю зависимость, и подумай, как ее замокать.
