# Неделя 6, День 1 — AI-инструменты и роли

## Зачем эта неделя
Эта неделя не про "поговорить об AI", а про рабочий режим:
- когда AI реально экономит время
- когда он опасен
- как проверять его результат
- как использовать его на задачах из этого репозитория

## Какие роли инструментов важно различать
- чат-ассистент: объяснить, придумать, набросать
- IDE-ассистент: быстро дописать шаблонный код рядом с человеком
- coding agent: читать проект, менять файлы, запускать проверки
- multi-agent workflow: разбирать несколько независимых задач параллельно

## Примеры инструментов
- `Codex`
- `Claude Code`
- `ChatGPT`
- `GitHub Copilot`

## Как смотреть на AI правильно
AI не заменяет инженера. Он ускоряет:
- ориентирование в кодовой базе
- черновую генерацию тестов
- поиск edge cases
- первичный code review
- разбор логов и traceback

Финальное решение, запуск и проверка остаются за человеком.
