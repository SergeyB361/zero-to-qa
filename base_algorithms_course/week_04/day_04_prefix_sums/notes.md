# Неделя 4, День 4 - Префиксные суммы

## Что это такое
Префиксная сумма в позиции `i` - это сумма всех элементов от начала массива до этой позиции.

Пример:

```python
items = [2, 5, 1, 3]
prefix = [2, 7, 8, 11]
```

## Зачем это нужно
Если нужно много раз отвечать на вопросы вида:
- какова сумма на отрезке `[l, r]`?

то пересчитывать сумму каждый раз с нуля невыгодно.

Префиксные суммы позволяют отвечать быстрее.

## Идея
Сначала один раз строим массив prefix.

После этого сумма на отрезке вычисляется через разность двух префиксов.

## Формула
Если `prefix[i]` хранит сумму элементов до `i`, то:
- сумма `[0, r]` = `prefix[r]`
- сумма `[l, r]` = `prefix[r] - prefix[l - 1]`

## Где это полезно
- range sum queries;
- анализ логов и метрик;
- подсчёт количества элементов на диапазонах;
- более сложные задачи на массивы.

## Типичные ошибки
1. Путать индекс массива и индекс префикса.
2. Плохо обработать случай `l == 0`.
3. Считать префикс заново для каждого запроса.
4. Не понимать, зачем нужен preprocessing.

## Практическое правило
Если есть много запросов к диапазонам, подумай о preprocessing.
Префиксные суммы - один из самых простых и полезных примеров.
