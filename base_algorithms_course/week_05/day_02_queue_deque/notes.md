# Неделя 5, День 2 - Очередь и deque

## Что такое очередь
Очередь - это структура данных по принципу **FIFO**:
- first in
- first out

Первый добавленный элемент извлекается первым.

## Базовые операции
- добавить в конец;
- взять из начала.

## Почему list не всегда хорош
Если использовать обычный список и делать `pop(0)`, это не лучший вариант для производительности.

Для очередей в Python обычно используют `collections.deque`.

## deque
`deque` = double-ended queue.

Она умеет эффективно:
- добавлять слева и справа;
- удалять слева и справа.

Полезные методы:
- `append()`
- `appendleft()`
- `pop()`
- `popleft()`

## Где очередь полезна
- обработка задач по порядку;
- симуляции клиентов и запросов;
- BFS по графу;
- окна фиксированного размера.

## Чем queue отличается от stack
- stack: последний пришёл - первый ушёл
- queue: первый пришёл - первый ушёл

## Типичные ошибки
1. Использовать список вместо `deque` для частых `pop(0)`.
2. Путать `appendleft` и `popleft`.
3. Путать очередь и стек.
4. Не обрабатывать пустую очередь.
