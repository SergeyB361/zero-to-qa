# Неделя 5, День 3 - Проверка скобок

## Почему здесь нужен стек
Когда видим открывающую скобку, её нужно запомнить.
Когда видим закрывающую, нужно проверить, что она закрывает последнюю незакрытую.

Это ровно поведение стека.

## Базовая идея
1. Идём по строке слева направо.
2. Открывающие скобки кладём в стек.
3. Закрывающая скобка требует взять верхушку стека.
4. Если типы не совпали или стек пуст - строка некорректна.
5. В конце стек должен быть пуст.

## Какие скобки обычно проверяют
- `()`
- `[]`
- `{}`

## Что важно
Проверка скобок - это классическая задача на стек.
Если ты её понимаешь, значит уже нормально чувствуешь идею LIFO.

## Типичные ошибки
1. Не проверять пустой стек перед `pop()`.
2. Проверять только количество скобок, а не порядок.
3. Забывать финальную проверку пустоты стека.
4. Неправильно сопоставлять типы скобок.
