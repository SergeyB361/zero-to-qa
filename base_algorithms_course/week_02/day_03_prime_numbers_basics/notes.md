# Неделя 2, День 3 - Простые числа: база

## Что такое простое число
Простое число - это число больше `1`, которое делится только на:
- `1`
- само себя

Примеры простых:
- `2`
- `3`
- `5`
- `7`
- `11`

## Что не является простым
- `1` не простое число
- числа вроде `4`, `6`, `8`, `9`, `10` составные

## Наивная проверка
Можно проверять делители от `2` до `n - 1`.

Это работает, но неэффективно.

## Улучшение
Если у числа есть делитель больше `sqrt(n)`, то второй делитель уже будет меньше `sqrt(n)`.

Значит достаточно проверять делители только до `sqrt(n)`.

На практике это сильное ускорение.

## Особый случай: число 2
`2` - простое и одновременно единственное чётное простое число.

## Где это нужно
- задачи на теорию чисел;
- генерация простых чисел;
- оптимизация перебора делителей;
- подготовка к решету Эратосфена.

## Типичные ошибки
1. Считать `1` простым.
2. Проверять делители до `n`, а не до `sqrt(n)`.
3. Не обрабатывать чётные числа отдельно.
4. Не учитывать маленькие `n` вроде `0`, `1`, `2`.

## Практическое правило
Для проверки одного числа на простоту чаще всего достаточно проверки делителей до `sqrt(n)`.

Если нужно проверить много чисел подряд, лучше использовать решето.
