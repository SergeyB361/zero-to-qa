# ТЗ: Number Toolkit

Реализуй проект в `number_toolkit.py`.

## Функции, которые должны быть в MVP
1. `gcd(a, b)`
2. `is_prime(n)`
3. `sieve(limit)`
4. `prime_factorization(n)`
5. `simplify_fraction(numerator, denominator)`

## Требования
- каждая функция решает одну задачу;
- используется подходящий алгоритм, а не случайный перебор;
- `main()` показывает минимум 5 демонстрационных вызовов;
- вывод читается глазами.

## Пример допустимого вывода
```text
GCD(48, 18): 6
Is prime(29): True
Primes up to 20: [2, 3, 5, 7, 11, 13, 17, 19]
Factorization of 84: [2, 2, 3, 7]
Simplified fraction 42/56: (3, 4)
```

## Критерии готовности
- файл запускается без ошибок;
- функции разбиты логически;
- в проекте реально используются темы недели;
- код читается и проверяется без лишней магии.
