# Неделя 2, День 7 - Мини-проект: Number Toolkit

## Цель дня
Собрать набор числовых алгоритмов в одном небольшом инструменте.

## Что ты собираешь
Файл с несколькими функциями, которые умеют:
- считать НОД;
- проверять простоту;
- строить список простых чисел до `n`;
- выполнять факторизацию;
- показывать demo flow через `main()`.

## Что важно
- каждая задача решается отдельной функцией;
- логика не смешивается в один блок;
- вывод читается без открытия кода.

## Следующий шаг
Открой `spec.md` и реализуй проект в `number_toolkit.py`.
